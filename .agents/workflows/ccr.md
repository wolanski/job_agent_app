# /ccr — Raise a Contract Change Request (CCR)

READ:
- process/PROGRESS.md
- product/contracts/*
- product/PRD.md (acceptance impact)

DO:
1) Create a new CCR-### entry in PROGRESS with:
   - problem statement
   - failing evidence (what broke / where)
   - minimal change proposal (what to change in product/contracts/*)
   - impact analysis (what code/stories are affected)
2) STOP and ask for a human decision (approve/reject/defer).
